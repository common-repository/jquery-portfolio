== Description ==

The jQuery Portfolio plugin allows you to quickly create an animated and sortable portfolio on your site.

Features:

* Custom Post Type for Portfolio Items
* Settings to control Portfolio Item Height and Width
* Various styles for the sorting options bar
* Easy development integration of custom options bar styles
* Portfolio images are displayed in an array and link to their respective description pages.
* Easily place your portfolio on any page using the shortcode "[portfolio]"
* Categorize a portfolio item in more than one category. 
* Awesome animated sorting by isotope.


== Installation ==

1. Upload `jquery-portfolio.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
